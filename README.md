# TMDb Movies Analysis 🎬

Exploratory data analysis project using the TMDb Movies dataset to investigate
which factors are associated with movie financial success.

## 📌 Project Overview

This project analyzes approximately 10,000 movies from **The Movie Database (TMDb)**.
The goal is to explore relationships between adjusted revenue and factors such as
budget, popularity, user ratings and vote counts.

The analysis was developed as part of a data analysis project and follows a
clean and well-documented data workflow.

## ❓ Research Questions

1. What factors are associated with higher adjusted movie revenue?
2. How do user ratings relate to a movie’s financial performance?
